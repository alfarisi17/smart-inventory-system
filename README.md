# Smart Inventory System

This project aims to create a smart inventory management system that provides users with real-time stock updates, automatic reordering, and data analytics.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js
- npm
- MongoDB

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/alfarisi17/smart-inventory-system.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the application
   ```sh
   npm start
   ```
